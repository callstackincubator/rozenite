### React Native DevTools Plugin Framework

[![mit licence][license-badge]][license] [![npm downloads][npm-downloads-badge]][npm-downloads] [![Chat][chat-badge]][chat] [![PRs Welcome][prs-welcome-badge]][prs-welcome]

Rozenite brings plug-and-play panels to React Native DevTools. Install plugins, enable them, and start debugging in minutes ‚Äî no extra windows, servers, or tabs. Works with Expo and bare React Native, across Metro and Re.Pack.

![Plugin slideshow](https://rozenite.dev/plugin-slideshow.webp)

## Features

- **Plug-and-Play Panels**: Install plugins and they auto-appear in DevTools ‚Äî start debugging in minutes.
- **Unified DevTools Experience**: All panels live inside React Native DevTools ‚Äî no extra windows, servers, or browser tabs.
- **Real-Time Debugging**: Live updates for network requests, storage and performance as you use your app.
- **Works With Your Setup**: Compatible with Expo and bare React Native projects. Supports Metro and Re.Pack based workflows.
- **Production-Safe Controls**: Plugins are automatically disabled in production builds ‚Äî no plugin code ships to your users.
- **Easy to Build Your Own**: When you need something custom, create your own panel with type-safe, isomorphic APIs and great DX.
