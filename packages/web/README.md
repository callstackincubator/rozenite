### Rozenite for Web

[![mit licence][license-badge]][license] [![npm downloads][npm-downloads-badge]][npm-downloads] [![Chat][chat-badge]][chat] [![PRs Welcome][prs-welcome-badge]][prs-welcome]

**Rozenite for Web** enables use of Rozenite and React Native DevTools in projects targeting the web platform.

The **Rozenite Chrome extension** is required for the integration to work correctly ‚Äî install it in your browser to connect to your React Native web app.
